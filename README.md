# unlumen UI Docs

Standalone open-source documentation app for unlumen UI.

## Development

```bash
pnpm install
pnpm dev
```

The docs app uses `basePath: "/docs"`, so local docs run at:

```txt
http://localhost:3001/docs
```

## Registry

Only public/free registry items belong in this repository. Premium component
source and private product APIs stay in `unlumen-ui-pv`.

```bash
pnpm registry:check
```
