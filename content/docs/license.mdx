- Use free components in any project, including paid client work
- Modify, refactor, and build on top of them
- Take inspiration from them to create your own components

**You cannot:**

- Sell or redistribute the components themselves as a standalone product, template pack, UI kit, or component library

## Pro Components

Pro components are available exclusively to licensed users.

**You can:**

- Use Pro components in any number of personal or commercial projects covered by your license
- Modify and adapt them to fit your project
- Use the same Polar license key to activate Pro on the website and authenticate Pro installs with the shadcn CLI

**You cannot:**

- Share, redistribute, or resell Pro component source code
- Use Pro components without a valid Unlumen UI Pro license

## In short

Use anything freely in your projects. Build on them, remix them, ship them to clients. The only restriction: don't repackage and sell them as-is.
